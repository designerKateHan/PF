Sound Board Web site design

A soundboard site is a site that can hear the corresponding sound when each button is pressed. I chose three movies and edited the sound of each one in the movie. All are designed with responsive design, while the desktop type is designed with a slightly more flexible grid system. In the mobile version, it is composed of sections of the same size, and the navigation is located at the bottom so that it can be easily pressed with the finger.
